# Course 1 — Supervised Machine Learning: Regression & Classification

**Instructor**: Andrew Ng (DeepLearning.AI / Stanford Online)
**Part of**: Machine Learning Specialization 
**Duration**: ~3 weeks, ~10 hours/week 
**Level**: Beginner / Intermediate  

---

## What You'll Learn

- How to build machine learning models in **Python** using core libraries like **NumPy** and **scikit-learn**.
- Supervised learning techniques for:
  - **Regression**: Using linear regression to predict continuous values, including working with multiple input features. 
  - **Classification**: Learning how to perform binary classification using **logistic regression**, and handling overfitting with **regularization**.
- Feature engineering—scaling, polynomial features, and vectorization—to improve model performance.
- Optimization techniques (e.g., gradient descent) for finding model parameters. 
- Regularization to avoid overfitting in logistic regression.

---

## Course Structure

| Week | Topics Covered |
|---|---|
| **Week 1** | Introduction to machine learning, core concepts, and problem framing
| **Week 2** | Regression: multiple input variables, feature scaling, polynomial regression, vectorization
| **Week 3** | Classification: logistic regression, regularization, implementation with scikit-learn

The course includes **9 assignments** (programming + quizzes) to reinforce learning.

## Skills Gained

- Predictive modeling using regression and classification

- Using Python, NumPy, and scikit-learn for machine learning workflows

## Why This Course Matters

Course 1 is foundational in the broader **Machine Learning Specialization**. By mastering regression and classification, you gain the core building blocks of supervised learning — essential for almost all machine learning applications. According to the specialization overview, this course balances **intuitive visual explanations**, **hands‑on coding**, and optional **mathematical deep dives**. 
Learning these concepts thoroughly will make subsequent courses (like neural networks, decision trees, and unsupervised learning) much easier.
